# excel-report-generator-improved
Last week I built a Python script that generates Excel reports.

But it only worked on one fixed dataset.

So I improved it.

Now it works on ANY CSV file (no matter the structure, no matter the columns).

💡Here's what it does automatically:
→ Detects numeric and category columns
→ Generates a clean summary (avg, max, min per column)
→ Builds a styled Excel report with charts
→ Handles messy data (nulls, inconsistent casing, mixed types)
